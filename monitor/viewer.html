mazg.ch - Viewer

This tool visualizes data recorded from one or more GPS/GNSS receivers. You can load multiple files, view their tracks together on an interactive map, and compare them in a chart. Offsets to a track named Reference are calculated for accuracy analysis. All processing happens in your browser, no data is uploaded to any server.

Tracks

Load multiple .ubx files recorderd with u-center or any other application that captures raw data from a GNSS/GPS receiver. This page uses .json environment file cointaining tracks, places and your configuration. For ease of use drag and drop files to this page. If you like to share your environment with a friend or colleages, simply downloaded environment file, and share a link to this page alongside.

  • Change the visibility: Click on the icons in column to change the visibility and apperance of the track.
  • Name tracks: Loaded tracks can be named individually. For the reference/truth track use the name Reference.
  • Set color: Use the color selectors to change the line color of any track.
  • Set reference: The reference track is used to calculate offests from that reference, this is usefull evaluate accuracies.
  • Adjust interval: Modify the UTC time interval to by adjusting the slider.

Map & Places

Each track is shown as a colored polyline, matching the track's assigned color. Fix quality is indicated by points along the track, click a point to view details. Info markers highlight locations where important errors, warnings or notifications were collected. You can zoom and pan the map using the mouse. Use the layer control in the top right to switch between base maps and toggle the visibility of places. Places help you define areas of interest that you like people to revisit.

  • Zoom to area: Hold Shift and drag to select an area to zoom to.
  • Define a place: Hold Ctrl and drag to to select an area to be defined as a new place.
  • Resize to place: Hold Ctrl and click on the dashed place rectangle in the map.
  • Fit to place: Choose a place from the list or click on a dashed place rectangle in the map.
  • Adjust brightness: Change the base layer opacity to improve visibility.
